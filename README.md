This simple recording application is originated from [fireShip IO](https://fireship.io/lessons/electron-screen-recorder-project-tutorial/).

But I tried to avoid using `nodeIntegration` rather than using it to make that application.

So, this app could be the hints for how to nourish your app using node module without having risk with `nodeIntegration`.
